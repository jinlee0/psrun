# psrun

## Installation
1. Clone Git
```bash
git clone https://github.com/jinlee0/psrun.git
cd psrun
```

2. Build Project
```bash
cargo build
```

1. 문제 풀이 중인 디렉토리에 input 파일을 생성하고 테스트 케이스들을 빈 줄로 구분하여 작성한다.
```
4
3
0
4
0

10
1
3
5
4
0
0
7
0
0
6

```
2. 해당 디렉토리에서 psrun을 실행한다.
```bash
psrun
```
